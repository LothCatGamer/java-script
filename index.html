To set up visual studio code, you need to go to this link -https://code.visualstudio.com/Download- and press the windows, mac or penquin option depending on what device your using. Then, when you open visual studio code you press the new file button (unless you want to open a file you have then press open and choose your file) and hover on top of open editors, press the first picture from the left and name your 'untitled folder 1' index.html if you want to code html, style.css to code css and java.script to code javascript. After that START CODING!!!

The console screen is where all the output from your code is displayed after you use console.log. You get the console screen when you right click on a website and press 'inspect'. It's usefull to see what the output is to see if your code is going well or not, and if it's doing what you want it to do.
